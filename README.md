# Ecommerce_analysis_sql
This project analyzes an e-commerce orders dataset to understand business performance across different dimensions such as payment methods, marketing channels, and coupon usage.  The goal is to identify patterns in revenue, order behavior, and cancellation rates.

Source: Kaggle – E-commerce Orders and Customer Dataset
https://www.kaggle.com/datasets/hammadansari7/e-commerce-orders-and-customer

This project was created as part of my portfolio to demonstrate SQL skills in business oriented data analysis.

**Key Analyses**
1. Profitability by Payment Method
Revenue per payment type
Cancellation rate
Average order value

**Key Insights:** credit card and online shoping generate higher revenue, but it should be noted that credit cards have the highest cancellation rate of all other payment sources.
